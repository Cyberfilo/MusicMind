    Per D-05, the endpoints are:

    a. `POST /api/claude/key` (store/update key, per D-01, D-06):
       - Accepts StoreKeyRequest body
       - Calls store_api_key(engine, encryption, user_id=current_user["user_id"], api_key=body.api_key)
       - Returns 201 with {"message": "API key stored"}
       - status_code=status.HTTP_201_CREATED

    b. `GET /api/claude/key/status` (check if configured, per D-02):
       - Calls get_api_key_status(engine, encryption, user_id=current_user["user_id"])
       - Returns KeyStatusResponse(**status_dict)

    c. `POST /api/claude/key/validate` (test key works, per D-03, D-04):
       - Calls get_decrypted_api_key(engine, encryption, user_id=current_user["user_id"])
       - If None: return ValidateKeyResponse(valid=False, error="No API key configured")
       - Else: calls validate_anthropic_key(api_key)
       - Returns ValidateKeyResponse(**result)

    d. `DELETE /api/claude/key` (remove key, per D-05):
       - Calls delete_api_key(engine, user_id=current_user["user_id"])
       - If not deleted: raise HTTPException(404, "No API key configured")
       - Returns {"message": "API key removed"}

    e. `GET /api/claude/key/cost` (cost estimate, per D-07, D-08):
       - Calls estimate_chat_cost()
       - Returns CostEstimateResponse(**cost_dict)

    Router: `router = APIRouter(prefix="/api/claude", tags=["claude"])`

       - Add line: `api_router.include_router(claude_router)`
