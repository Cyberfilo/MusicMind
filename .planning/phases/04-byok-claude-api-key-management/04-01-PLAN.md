    Create `backend/src/musicmind/api/claude/service.py` with these functions following the Phase 3 service.py pattern (async DB operations with engine parameter, EncryptionService for encrypt/decrypt):

    1. `mask_api_key(api_key: str) -> str` — Pure function. Returns "sk-ant-...{last4}" where last4 is the last 4 characters of the key. If key is shorter than 8 chars, return "****".

    2. `async def store_api_key(engine, encryption: EncryptionService, *, user_id: str, api_key: str) -> None` — Per D-01, D-06. Encrypts key via encryption.encrypt(api_key). Uses dialect-agnostic SELECT-then-INSERT/UPDATE (same pattern as upsert_service_connection in Phase 3). Updates updated_at on overwrite. Service column defaults to "anthropic".

    3. `async def get_api_key_status(engine, encryption: EncryptionService, *, user_id: str) -> dict` — Per D-02. SELECT api_key_encrypted WHERE user_id. If no row: return {"configured": False, "masked_key": None, "service": "anthropic"}. If row exists: decrypt key, call mask_api_key, return {"configured": True, "masked_key": masked, "service": "anthropic"}.

    4. `async def get_decrypted_api_key(engine, encryption: EncryptionService, *, user_id: str) -> str | None` — SELECT api_key_encrypted WHERE user_id. Decrypt and return plaintext, or None if no row.

    5. `async def delete_api_key(engine, *, user_id: str) -> bool` — DELETE FROM user_api_keys WHERE user_id AND service="anthropic". Return rowcount > 0.

    6. `async def validate_anthropic_key(api_key: str) -> dict` — Per D-03. Create `anthropic.AsyncAnthropic(api_key=api_key)`. Call `client.messages.create(model="claude-sonnet-4-20250514", max_tokens=1, messages=[{"role": "user", "content": "hi"}])`. On success return {"valid": True, "error": None}. Catch `anthropic.AuthenticationError` -> {"valid": False, "error": "Invalid API key"}. Catch `anthropic.APIError` as e -> {"valid": False, "error": str(e)}. Catch `Exception` as e -> {"valid": False, "error": f"Validation failed: {e}"}.

    7. `def estimate_chat_cost() -> dict` — Per D-07, D-08. Return static dict: {"model": "claude-sonnet-4-20250514", "estimated_cost_per_message": "$0.01-0.05", "input_token_price": "$3.00 / 1M tokens", "output_token_price": "$15.00 / 1M tokens"}. This is a pure function with hardcoded current pricing, not real-time tracking.

    Conventions: `from __future__ import annotations` at top, module docstring, logger = logging.getLogger(__name__), type hints on all functions, use sa.and_() for compound WHERE clauses.
