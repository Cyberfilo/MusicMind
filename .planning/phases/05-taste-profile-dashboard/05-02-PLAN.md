**1. Create TasteService** at `backend/src/musicmind/api/taste/service.py`:

Per D-04: When user requests their taste profile, the backend: (1) checks if cached profile is fresh (< 24h), (2) if stale, fetches from service API, (3) caches raw data, (4) runs build_taste_profile, (5) returns structured JSON.

```python
"""Taste profile pipeline: staleness check, fetch, cache, compute, return."""
from __future__ import annotations

import json
import logging
from datetime import UTC, datetime, timedelta
from typing import Any

import sqlalchemy as sa

from musicmind.api.services.service import (
    generate_apple_developer_token,
    get_user_connections,
    refresh_spotify_token,
    upsert_service_connection,
)
from musicmind.api.taste.fetch import (
    enrich_spotify_genres,
    fetch_apple_music_library,
    fetch_apple_music_recently_played,
    fetch_spotify_recently_played,
    fetch_spotify_saved_tracks,
    fetch_spotify_top_artists,
    fetch_spotify_top_tracks,
)
from musicmind.db.schema import (
    listening_history,
    service_connections,
    song_metadata_cache,
    taste_profile_snapshots,
)
from musicmind.engine.profile import build_taste_profile
from musicmind.security.encryption import EncryptionService

logger = logging.getLogger(__name__)

STALENESS_HOURS = 24  # Per D-06
```

Class `TasteService` with these methods:

a) `async def get_profile(self, engine, encryption: EncryptionService, settings, *, user_id: str, service: str | None = None, force_refresh: bool = False) -> dict[str, Any]`:
   - Resolve service: if None, pick first connected service via get_user_connections (per D-08)
   - If no connected service found, raise ValueError("No connected service found")
   - Check fresh snapshot: query taste_profile_snapshots WHERE user_id AND service_source AND computed_at >= (now - 24h). If found and not force_refresh, return snapshot data as dict.
   - If stale or force_refresh: call _fetch_and_cache_data, then _compute_and_save_profile
   - Return profile dict

b) `async def _resolve_service(self, engine, *, user_id: str, service: str | None) -> str`:
   - If service is provided and not None, return it
   - Otherwise query get_user_connections, return the first one's "service" field
   - Raise ValueError if none found

c) `async def _get_fresh_snapshot(self, engine, *, user_id: str, service_source: str) -> dict[str, Any] | None`:
   - Query taste_profile_snapshots WHERE user_id AND service_source AND computed_at >= cutoff
   - ORDER BY computed_at DESC LIMIT 1
   - Return dict from row._mapping or None
   - Use UTC datetime for cutoff computation
   - Normalize cutoff for SQLite: strip tzinfo when comparing with timezone-naive stored dates (per Phase 03 convention: "UTC normalization in router for SQLite timezone-naive datetime compat")

d) `async def _fetch_and_cache_data(self, engine, encryption: EncryptionService, settings, *, user_id: str, service: str) -> tuple[list[dict], list[dict]]`:
   - Get connection from DB: query service_connections WHERE user_id AND service
   - Decrypt access token via encryption.decrypt(row.access_token_encrypted)
   - **For Spotify:**
     - Check token_expires_at. If expired or within 60s, refresh via refresh_spotify_token (per Pitfall 4). Decrypt refresh_token, call refresh_spotify_token with settings.spotify_client_id. If refreshed, update connection in DB via upsert_service_connection.
     - Call fetch_spotify_top_tracks, fetch_spotify_top_artists, fetch_spotify_saved_tracks, fetch_spotify_recently_played
     - Call enrich_spotify_genres(top_tracks + saved_tracks, artists)
     - songs = top_tracks + saved_tracks (deduplicated by catalog_id)
     - history = recently_played
   - **For Apple Music (per Pitfall 3):**
     - Generate developer_token via generate_apple_developer_token(settings.apple_team_id, settings.apple_key_id, settings.apple_private_key_path)
     - Decrypt music_user_token (stored as access_token in service_connections)
     - Call fetch_apple_music_library(developer_token, music_user_token)
     - Call fetch_apple_music_recently_played(developer_token, music_user_token)
     - songs = library result, history = recently_played result
   - Cache songs: for each song, do dialect-agnostic SELECT-then-INSERT/UPDATE into song_metadata_cache (user_id=user_id, service_source=service)
   - Cache history: INSERT into listening_history for each entry
   - Return (songs, history)

e) `async def _compute_and_save_profile(self, engine, *, user_id: str, service: str, songs: list[dict], history: list[dict]) -> dict[str, Any]`:
   - Call build_taste_profile(songs, history, use_temporal_decay=True) per D-12
   - Save to taste_profile_snapshots: INSERT with user_id, service_source=service, computed_at=now(UTC), plus all profile fields (genre_vector as JSON, top_artists as JSON, etc.)
   - Return profile dict with added "service" and "computed_at" keys

```python
"""Taste profile API endpoints."""
from __future__ import annotations

import logging
from datetime import UTC, datetime

from fastapi import APIRouter, Depends, HTTPException, Query, Request, status

from musicmind.api.taste.schemas import (
    AudioTraitsResponse,
    ArtistEntry,
    GenreEntry,
    TasteProfileResponse,
    TopArtistsResponse,
    TopGenresResponse,
)

router = APIRouter(prefix="/api/taste", tags=["taste"])

```

Endpoints:

a) `GET /api/taste/profile` -> TasteProfileResponse:
   - Params: service: str | None = Query(default=None), refresh: bool = Query(default=False)
   - Depends: user = Depends(get_current_user)
   - Call taste_service.get_profile(request.app.state.engine, request.app.state.encryption, request.app.state.settings, user_id=user["user_id"], service=service, force_refresh=refresh)
   - On ValueError (no service): return 400
   - On success: return TasteProfileResponse built from profile dict
   - Map top_artists from list[dict] to list[ArtistEntry]
   - computed_at: if datetime, convert to ISO string. If not present, use datetime.now(UTC).isoformat()

b) `GET /api/taste/genres` -> TopGenresResponse:
   - Same params + depends
   - Get profile, extract genre_vector
   - Return TopGenresResponse with genres=[GenreEntry(genre=k, weight=v) for k,v in genre_vector items sorted by value desc]

c) `GET /api/taste/artists` -> TopArtistsResponse:
   - Same params + depends
   - Get profile, extract top_artists
   - Return TopArtistsResponse with artists=[ArtistEntry(**a) for a in top_artists]

d) `GET /api/taste/audio-traits` -> AudioTraitsResponse:
   - Same params + depends
   - Get profile, extract audio_trait_preferences
   - If empty and service == "spotify": set note = "Audio traits not available for Spotify (Apple Music only)"
   - Return AudioTraitsResponse

All endpoints: wrap taste_service.get_profile in try/except ValueError -> HTTPException(400). Wrap unexpected exceptions -> HTTPException(500) with logger.exception.
