    **File 2 — `backend/src/musicmind/auth/service.py`:**
    Module docstring: `"""Auth service: password hashing, JWT token creation, and cookie helpers."""`

    Imports: `from __future__ import annotations`, `uuid`, `datetime`/`timedelta`/`timezone` from datetime, `bcrypt`, `jwt` (PyJWT), `from fastapi import Response`.

    Constants:
    - `ACCESS_TOKEN_EXPIRE_MINUTES = 30` (per D-02)
    - `REFRESH_TOKEN_EXPIRE_DAYS = 7` (per D-02)
    - `JWT_ALGORITHM = "HS256"` (per D-10)

    Functions (all module-level, no class):

    `hash_password(password: str) -> str`:
    - `bcrypt.hashpw(password.encode(), bcrypt.gensalt()).decode()` (per D-05, D-07)
    - Docstring: `"""Hash a password using bcrypt."""`

    `verify_password(password: str, hashed: str) -> bool`:
    - `bcrypt.checkpw(password.encode(), hashed.encode())` (per D-05)
    - Docstring: `"""Verify a password against its bcrypt hash."""`

    `create_access_token(user_id: str, email: str, *, secret_key: str, expires_delta: timedelta | None = None) -> str`:
    - Payload: `{"sub": user_id, "email": email, "iat": now_utc, "exp": now_utc + delta, "type": "access"}` (per D-10)
    - Default delta: 30 minutes (per D-02)
    - `jwt.encode(payload, secret_key, algorithm=JWT_ALGORITHM)`
    - Docstring: `"""Create a signed JWT access token."""`

    `create_refresh_token(user_id: str, *, secret_key: str, expires_delta: timedelta | None = None) -> tuple[str, str]`:
    - `token_id = str(uuid.uuid4())`
    - Payload: `{"sub": user_id, "jti": token_id, "iat": now_utc, "exp": now_utc + delta, "type": "refresh"}`
    - Default delta: 7 days (per D-02)
    - Returns `(jwt_token, token_id)` — token_id is stored in DB for revocation (per D-02, D-04)
    - Docstring: `"""Create a refresh token. Returns (jwt_token, token_id) for DB storage."""`

    `set_auth_cookies(response: Response, access_token: str, refresh_token: str, *, secure: bool = True) -> None`:
    - Sets `access_token` cookie: httponly=True, secure=secure, samesite="lax", max_age=30*60, path="/" (per D-01, D-03)
    - Sets `refresh_token` cookie: httponly=True, secure=secure, samesite="lax", max_age=7*24*60*60, path="/api/auth/refresh" (per D-03, scoped path per research pitfall 3)
    - `secure` parameter defaults True; caller passes `not settings.debug` for local dev (per research pitfall 7)
    - Docstring: `"""Set access and refresh tokens as httpOnly cookies."""`

    `clear_auth_cookies(response: Response) -> None`:
    - `response.delete_cookie("access_token", path="/")` (per D-04)
    - `response.delete_cookie("refresh_token", path="/api/auth/refresh")` (per D-04)
    - Docstring: `"""Clear auth cookies on logout."""`

    **File 4 — `backend/src/musicmind/auth/dependencies.py`:**
    Module docstring: `"""FastAPI dependencies for authentication."""`

    Imports: `from __future__ import annotations`, `from fastapi import HTTPException, Request, status`, `import jwt`.

    `async def get_current_user(request: Request) -> dict[str, str]`:
    - Extract `access_token` from `request.cookies.get("access_token")`
    - If missing: raise HTTPException 401 "Not authenticated"
    - Decode with `jwt.decode(token, request.app.state.settings.jwt_secret_key, algorithms=["HS256"])` (MUST specify algorithms per research pitfall 2)
    - If `payload.get("type") != "access"`: raise HTTPException 401 "Invalid token type"
    - Return `{"user_id": payload["sub"], "email": payload["email"]}`
    - Catch `jwt.ExpiredSignatureError`: raise HTTPException 401 "Token expired"
    - Catch `jwt.InvalidTokenError`: raise HTTPException 401 "Invalid token"
    - Docstring: `"""Extract current user from access token cookie."""`
