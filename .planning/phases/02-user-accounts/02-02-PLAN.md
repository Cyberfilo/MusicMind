    **Endpoint 1 — POST /api/auth/signup (per D-13, D-14, ACCT-01):**
    ```python
    @router.post("/signup", status_code=status.HTTP_201_CREATED)
    async def signup(request: Request, response: Response, body: SignupRequest) -> dict:
    ```
    - Get engine and settings from `request.app.state`
    - Hash password: `hash_password(body.password)` (per D-05, D-07)
    - Generate user_id: `str(uuid.uuid7())` (per research — Python 3.14 has uuid7)
    - In `async with engine.begin() as conn:`:
      - Check duplicate email: `SELECT id FROM users WHERE email = body.email`
      - If exists: raise HTTPException 400 "Account creation failed" (per D-14 — generic error)
      - Insert user: `users.insert().values(id=user_id, email=body.email, password_hash=password_hash, display_name=body.display_name or body.email.split("@")[0])`
    - Create access token: `create_access_token(user_id, body.email, secret_key=settings.jwt_secret_key)`
    - Create refresh token: `create_refresh_token(user_id, secret_key=settings.jwt_secret_key)` returns `(token, token_id)`
    - Store refresh token in DB: `refresh_tokens.insert().values(id=token_id, user_id=user_id, expires_at=now+7days)`
    - Set cookies: `set_auth_cookies(response, access_token, refresh_token, secure=not settings.debug)` (per D-01, D-03, research pitfall 7)
    - Return `{"user_id": user_id, "email": body.email, "message": "Account created"}`

    **Endpoint 2 — POST /api/auth/login (per ACCT-02):**
    ```python
    @router.post("/login")
    async def login(request: Request, response: Response, body: LoginRequest) -> dict:
    ```
    - Fetch user by email: `SELECT id, email, password_hash, display_name FROM users WHERE email = body.email`
    - If not found OR `verify_password(body.password, user.password_hash)` is False: raise HTTPException 401 "Invalid credentials" (generic per D-14 spirit)
    - Create access + refresh tokens (same as signup)
    - Store refresh token in DB
    - Set cookies (same as signup)
    - Return `{"user_id": user.id, "email": user.email, "message": "Login successful"}`

    **Endpoint 3 — POST /api/auth/logout (per D-04, ACCT-03):**
    ```python
    @router.post("/logout")
    async def logout(request: Request, response: Response) -> dict:
    ```
    - Extract refresh token from `request.cookies.get("refresh_token")`
    - If refresh token exists, decode it (ignore expiry errors) to get `jti` and revoke in DB: `UPDATE refresh_tokens SET revoked=true WHERE id=jti`
    - Clear cookies: `clear_auth_cookies(response)` (per D-04)
    - Return `{"message": "Logged out"}`

    **Endpoint 4 — POST /api/auth/refresh (per D-11, ACCT-02):**
    ```python
    @router.post("/refresh")
    async def refresh(request: Request, response: Response) -> dict:
    ```
    - Extract refresh token from `request.cookies.get("refresh_token")`
    - If missing: raise HTTPException 401 "No refresh token"
    - Decode with PyJWT: `jwt.decode(token, settings.jwt_secret_key, algorithms=["HS256"])`
    - Verify `payload.get("type") == "refresh"`, else 401
    - Check DB: `SELECT id, revoked FROM refresh_tokens WHERE id = payload["jti"]`
    - If not found or revoked: raise HTTPException 401 "Invalid refresh token"
    - Revoke the old refresh token in DB (token rotation)
    - Create new access + refresh tokens
    - Store new refresh token in DB
    - Set new cookies
    - Return `{"user_id": payload["sub"], "message": "Token refreshed"}`

    **Endpoint 5 — GET /api/auth/me (protected, per ACCT-02):**
    ```python
    @router.get("/me")
    async def me(request: Request, current_user: dict = Depends(get_current_user)) -> dict:
    ```
    - Fetch user details from DB: `SELECT id, email, display_name FROM users WHERE id = current_user["user_id"]`
    - If not found: raise HTTPException 404 "User not found"
    - Return `{"user_id": user.id, "email": user.email, "display_name": user.display_name}`

    ```python
    _settings = Settings()

    @asynccontextmanager
    async def lifespan(app: FastAPI) -> AsyncIterator[None]:
        settings = _settings
        engine = create_engine(settings.database_url, echo=settings.debug)
        encryption = EncryptionService(settings.fernet_key)
        app.state.engine = engine
        app.state.settings = settings
        app.state.encryption = encryption
        yield
        await engine.dispose()

    app = FastAPI(title="MusicMind", version="0.1.0", lifespan=lifespan)
    app.include_router(api_router)
    app.add_middleware(
        CSRFMiddleware,
        secret=_settings.jwt_secret_key,
        sensitive_cookies={"access_token", "refresh_token"},
        cookie_name="csrftoken",
        header_name="x-csrf-token",
    )
    ```
