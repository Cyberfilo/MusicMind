    ```
    class ChatService:
        MAX_TOOL_CALLS = 10  # per D-17
        CONTEXT_WINDOW_MESSAGES = 20  # per D-09
        MODEL = "claude-sonnet-4-20250514"

        async def send_message(
            self,
            engine,
            encryption,
            settings,
            *,
            user_id: str,
            conversation_id: str | None,
            message: str,
        ) -> AsyncGenerator[dict, None]:
            """Send a message and yield SSE events as the response streams."""
    ```

    Implementation details:

    1. **BYOK key retrieval**: Call get_decrypted_api_key(). If None, yield error event "No API key configured" and return.

    2. **Conversation load/create**:
       - If conversation_id is None: generate uuid7, insert new row in chat_conversations with title = message[:50], yield {"event": "conversation_id", "data": {"id": new_id}}.
       - If conversation_id is provided: load existing conversation from DB, extract messages JSON.

    3. **Context window management** (per D-09):
       - Take last CONTEXT_WINDOW_MESSAGES from conversation messages.
       - Append the new user message.
       - Convert to Anthropic message format (role: user/assistant, content blocks).

    4. **System prompt construction**:
       - Include: "You are MusicMind, a music discovery assistant. You have access to the user's music library and taste profile across their connected services."
       - Include user's connected services (query service_connections table).
       - Include brief taste summary if available (query latest taste_profile_snapshots).
       - Include available tool descriptions summary.
       - Per CONTEXT specifics: include user's top 3 genres.

    5. **Per-request AsyncAnthropic client** (per D-03):
       ```python
       client = anthropic.AsyncAnthropic(api_key=api_key)
       ```

    6. **Agentic loop** (per D-01):
       ```
       tool_call_count = 0
       while True:
           response = await client.messages.create(
               model=self.MODEL,
               max_tokens=4096,
               system=system_prompt,
               messages=anthropic_messages,
               tools=TOOL_DEFINITIONS,
               disable_parallel_tool_use=True,  # per D-06
               stream=True,  # SSE streaming per D-02
           )

           # Process streamed response
           async with response as stream:
               async for event in stream:
                   if event is text_delta:
                       yield {"event": "text", "data": {"text": delta.text}}
                   if event is tool_use:
                       # Yield tool_start (per D-11)
                       yield {"event": "tool_start", "data": {"tool": tool_name, "input": tool_input}}
                       # Execute tool
                       result = await self._execute_tool(tool_name, engine, encryption, settings, user_id, tool_input)
                       # Yield tool_end (per D-11)
                       yield {"event": "tool_end", "data": {"tool": tool_name, "result": result}}
                       tool_call_count += 1

           # Check stop reason
           if stop_reason == "end_turn" or stop_reason == "max_tokens":
               break
           if stop_reason == "tool_use" and tool_call_count < MAX_TOOL_CALLS:
               # Append assistant message + tool_result to anthropic_messages
               # Continue loop
               continue
           else:
               break  # Hit cap or unexpected stop
       ```

       Use the Anthropic SDK's streaming API. The stream yields events like:
       - `RawContentBlockStartEvent` with `content_block.type == "tool_use"`
       - `ContentBlockDeltaEvent` with `delta.type == "text_delta"`
       - `MessageStopEvent`

       Handle the streaming events correctly to extract text deltas and tool_use blocks.

    7. **Tool execution** via _execute_tool():
       ```python
       async def _execute_tool(self, tool_name, engine, encryption, settings, user_id, tool_input):
           executor = TOOL_EXECUTORS.get(tool_name)
           if not executor:
               return {"error": f"Unknown tool: {tool_name}"}
           try:
               return await executor(engine=engine, encryption=encryption, settings=settings, user_id=user_id, **tool_input)
           except Exception as e:
               return {"error": str(e)}
       ```

    8. **Error handling** (per D-14, D-15, D-16):
       Wrap the client.messages.create call in try/except:
       - anthropic.AuthenticationError -> yield error "API key expired or invalid. Please update your key in settings."
       - anthropic.RateLimitError -> yield error "Rate limit reached. Please wait a moment and try again."
       - anthropic.APIStatusError with status 402 or message containing "balance" -> yield error "Insufficient API balance. Please check your Anthropic account."
       - Other anthropic.APIError -> yield error with sanitized message.

    9. **Conversation persistence**:
       After the agentic loop completes (or errors), update chat_conversations.messages with the full message history (user message + assistant response + any tool calls/results).
       Update updated_at timestamp.

    10. **Yield "done" event** at the end: {"event": "done", "data": {}}
