    1. Create backend/src/musicmind/api/chat/router.py with 4 endpoints (per D-12, D-13):

       ```python
       router = APIRouter(prefix="/api/chat", tags=["chat"])
       ```

       a) **POST /api/chat/message** (per D-12):
          - Accept: SendMessageRequest body
          - Auth: Depends(get_current_user)
          - Create ChatService instance
          - Call send_message() which returns an async generator of SSE events
          - Wrap in StreamingResponse with media_type="text/event-stream"
          - SSE format: each event yielded as `event: {type}\ndata: {json}\n\n`
          - Use a helper async generator that formats each dict from ChatService into SSE text:
            ```python
            async def _sse_generator(events):
                async for event in events:
                    event_type = event.get("event", "message")
                    data = json.dumps(event.get("data", {}))
                    yield f"event: {event_type}\ndata: {data}\n\n"
            ```
          - Return StreamingResponse(_sse_generator(chat_service.send_message(...)), media_type="text/event-stream")
          - Do NOT use sse-starlette library -- use plain StreamingResponse with manual SSE formatting (simpler, no extra dependency needed since starlette handles this natively).

       b) **GET /api/chat/conversations** (per D-13):
          - Auth: Depends(get_current_user)
          - Query chat_conversations table for user_id, ordered by updated_at DESC
          - Return ConversationListResponse with conversation metadata (id, title, message count, timestamps)

       c) **GET /api/chat/conversations/{conversation_id}** (per D-13):
          - Auth: Depends(get_current_user)
          - Query chat_conversations by id AND user_id (user isolation)
          - 404 if not found
          - Return ConversationResponse with full messages

       d) **DELETE /api/chat/conversations/{conversation_id}** (per D-13):
          - Auth: Depends(get_current_user)
          - Delete from chat_conversations where id AND user_id match
          - 404 if no rows deleted (conversation doesn't exist or belongs to another user)
          - Return {"message": "Conversation deleted"}
